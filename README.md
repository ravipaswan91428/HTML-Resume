# HTML-Resume project
## What is this ?
This is a simple resume landing page which is made by using HTML and CSS.
This shows the data of your like your qualification, skills, experience etc.

## Content
This page contains :-
- Header : This is the header of the page which shows your name and profession
- Main : This is a main section or you can also call it as a main cotent of the page. This page contains few section.
- Section: This is the sections of the main body which contains about you and your skill, experiences, projects, education and contact details.
- Table: This page also contains tables which make your details more classiffied.
